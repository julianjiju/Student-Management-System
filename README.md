# Student Management System

A Student Management CRUD web application built with ASP.NET Core MVC, Entity Framework Core and SQL Server. Supports Create, Read, Update and Delete operations on student records with a clean custom UI.

## Tech Stack

- **Framework:** ASP.NET Core MVC (.NET 8)
- **ORM:** Entity Framework Core
- **Database:** SQL Server
- **Frontend:** Razor Views, Bootstrap, Custom CSS
- **Language:** C#

---

## Features

- View all enrolled students in a clean table layout
- Add new students with form validation
- Edit existing student records
- View detailed info for each student
- Delete students with a confirmation step
- Enrollment date displayed without timestamp
- Responsive design with custom styling
